Agent Bees - ESP8266

The AB_ESP8266 ESP8266 WiFi interface for Xbee Sockets allows the ESP8266 to be used with Arduino FIO and other XBee based platforms.

AB_ESP8266 Features:

  • ESP8266 WiFi SoC
  • XBee Socket Form Factor
  • Jumper straps for reprogramming
  • LEDs for status indictors

About AB_ESP8266:

"AB_ESP8266" is a very quick hack to put the popular ESP8266 WiFI interface on the XBee form factor.

AB_ESP8266 Application Examples:

  • WiFi Sensor
  • WiFi Bridge

Using the AB_ESP8266 with the Teensy 3.1

There are several XBee Adaptor boards available for the Teensy 3.1 Plaform, such as this one from Sparkfun.

Okay, how can I get one ?

  • AB_ESP8266 is an open source hardware, DIY project.
  • It is designed to be easily soldered and hand-built as a DIY project.
  • You can order your own PCBs at Oshpark.com
  • Use this file to order the AB-ESP8266 PCB
  • If we get enough interest/requests, we will make it a complete product. Use the comments below if interested.
