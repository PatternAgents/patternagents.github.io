The AB_ESP8266 WiFi Interface for XBEE, allows the ESP8266 to be used with XBEE based platforms.
Using a USB-to-Serial XBEE adaptor, the AB_ESP8266 WiFi Interface is compatible with the nodeMCU platform.

{{page.product_slug}} Features:

  • ESP8266 WiFi System-on-Chip
  • Program using the Arduino IDE
  • Program using the Node.js
  • Program using the eLua
  • XBee Socket Form Factor
  • Flash programming using RTS/DTR, compatible with nodeMCU
  • Inter-Integrated Circuit(I2C) Bus supported
  • Serial-Parallel Interface(SPI) Bus supported

About {{page.product_slug}}:

"AB_ESP8266" is a quick hack to enable using the popular ESP8266 WiFi module with the XBee form factor.

{{page.product_slug}} Application Examples:

  • WiFi Sensor
  • WiFi Bridge

WiFi to/from Bluetooth Example (breadboard)

A WiFi to/from Bluetooth Bridge with FRAM memory sensor log wired on bread board. Because the XBee form factor uses 2.00MM connector spacing, while a standard breadboard is 2.54MM spacing, a Sparkfun XBee Breakout Board is used underneath the Agent Bee AB_ESP8266.
